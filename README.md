# SSA Dietary Diversity

This repository contains template R scripts illustrating the analytical workflow used in the manuscript:

**Climate variability, forest cover, cropland expansion, and child dietary diversity across Sub-Saharan Africa.**

## Repository Structure

- R/
  - Regression_Analysis_Template.R
  - Caterpillar_Plot_Template.R
  - Coefficient_Comparison_Template.R
  - Supplementary_Table_Template.R

- data/
  - Placeholder for input datasets.

- outputs/
  - Generated figures and tables.

## Data

The original DHS survey data and other input datasets are publicly available from the sources described in the manuscript and are not included in this repository.

Users should replace the placeholder file paths in the template scripts with their own data locations.
